# LOAD_redo
New LOAD files for UCSC genome browser visualization

trackline for loading into UCSC genome browser

```
track type=bigLolly name="DMPs" description="description" bigDataUrl=https://raw.githubusercontent.com/pebergmann/LOAD_redo/main/ADCU.bb lollySizeField=lollySize lollyField=lollyField visibility=full
```

trackline for enhancer/promoter visualization
```
track type=bigInteract name="PCHi-C interactions" description="A bigInteract file" useScore=off bigDataUrl=https://raw.githubusercontent.com/pebergmann/LOAD_redo/main/interact.bb visibility=full 
```
